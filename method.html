📚 方法说明

评分体系 · 计算逻辑 · 数据来源

🎯 评分体系（总分100分）

维度 分值 指标说明
财务健康 30分 ROE（净资产收益率） - 衡量股东权益回报水平
净利率 - 企业盈利能力核心指标
流动比率 - 短期偿债能力
资产负债率 - 财务杠杆水平
估值水平 30分 PE（市盈率） - 相对估值基础指标
EV/EBITDA、EV/Revenue、P/B - 多维度估值
分析师目标价上行空间 - 市场预期
新闻情绪 25分 媒体舆情分析 - 基于公开报道综合评估市场情绪（当前默认中性12分，待接入API）
催化剂 15分 营收增速、净利增速、自由现金流增速 - 成长性驱动因素

📈 交易建议规则

强力买入 总分 ≥75分 且 上行空间 >15%

高评分 + 高上行空间，建议积极建仓

买入 总分 ≥60分 且 上行空间 >5%

基本面良好，具备一定上涨潜力

持有 其他情况

建议继续观察，等待更好时机

卖出 总分 ≤35分 或 上行空间 <-10%

风险较高，建议及时止损或止盈

🧮 详细计算公式

财务健康得分 (30分)

财务得分 = ROE得分(0-8) + 净利率得分(0-8) + 流动比率得分(0-7) + 资产负债率得分(0-7)
  • ROE: >20%得8分，15-20%得6分，10-15%得4分，5-10%得2分，<5%得0分
  • 净利率: >20%得8分，15-20%得6分，10-15%得4分，5-10%得2分，<5%得0分
  • 流动比率: >2得7分，1.5-2得5分，1-1.5得3分，<1得0分
  • 资产负债率: <30%得7分，30-50%得5分，50-70%得3分，>70%得0分

估值水平得分 (30分)

估值得分 = PE得分(0-10) + EV/EBITDA得分(0-7) + 目标价空间得分(0-13)
  • PE: 低于行业平均30%以上得10分，低于行业平均得7分，高于行业平均得4分，远高于行业平均得0分
  • EV/EBITDA: 合理范围得7分，略高得4分，远高于得0分
  • 目标价上行空间: >30%得13分，20-30%得10分，10-20%得7分，0-10%得4分，<0%得0分

新闻情绪得分 (25分)

情绪得分 = 12分(基准) + 正面报道加分 - 负面报道减分

注：当前版本使用默认中性分数，正式版将接入新闻API进行实时舆情分析

催化剂得分 (15分)

催化剂得分 = 营收增速(0-5) + 净利增速(0-5) + 自由现金流增速(0-5)
  • 营收增速: >30%得5分，20-30%得4分，10-20%得3分，0-10%得2分，<0%得0分
  • 净利增速: >30%得5分，20-30%得4分，10-20%得3分，0-10%得2分，<0%得0分
  • 自由现金流: >20%得5分，10-20%得4分，0-10%得2分，<0%得0分
